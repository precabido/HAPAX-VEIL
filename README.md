# HAPAX-VEIL
Plataforma self-hosted de mensajes, secretos y archivos cifrados con enlaces autodestructivos. Cifrado lado cliente, servidor sin acceso al contenido, expiración configurable, burn-after-read, passphrase opcional, privacidad por diseño, anti-preview, adjuntos cifrados y arquitectura minimalista orientada a seguridad real.
